I am a streamer. How could use this tools ?

Enter your Chess.com name, select bullet, blitz or rapid and 'All' if you want your stats from the beginning or Session if you want to see your stats of your session. If your launch the page at 1 PM, each games finish after 1 PM will count to your session stats

I am a streamer. How could use this tools ?

This tools is build for you. Do the first part and copy the full link. Then go to your streaming software.
I will take OBS Studio as example.
Scenes > Sources -> Add -> Browser
Paste the URL.
Custom CSS is important.The default preset on OBS is : body { background-color: rgba(0, 0, 0, 0); margin: 0px auto; overflow: hidden; }
background-color: rgba(0, 0, 0, 0); means the background will be transparent. By default the the color of the text is Black.You may let it blakc or change it to White by adding color:white; The configuration for CSS may look like : 


        body { 
            background-color: rgba(0, 0, 0, 0); 
            margin: 0px auto; 
            overflow: hidden; 
            color: white;
            font-size: 48px;
        }
If you want to change CSS of Win / Draw / Loss , use .win .draw .loss CSS classes
